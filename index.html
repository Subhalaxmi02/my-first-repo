Pointers for block

  1. width of a block level element is 100% of the viewport width
  2. Height of a block is not set by default
  3. Height and width can be explicitely set using CSS

Pointers for inline

  1. width of an inline element is dependent on the content
  2. Height of a inline is not set by default
  3. Height and width can't be explicitely set using CSS
